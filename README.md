

# ela-fe

## dockerized run

### build and run docker image
```
yarn runDockerLocal
```
this will build project and create images and run the container

### stop and destroy image
```
yarn stopDockerLocal
```
this is needed before rebuilding the container

## Project setup
```
yarn install
```

